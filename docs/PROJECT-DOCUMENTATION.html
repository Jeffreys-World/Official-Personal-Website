1. Project Overview

A single-page developer portfolio supporting Jeffrey's pivot into software engineering and AI product building. The design is structurally inspired by haoqi.design — typography-led hero, minimal utilitarian navigation, list-based project rows, and technical "HUD" metadata — rebuilt with original content, copy, and a New York City visual identity.

The site was planned from two briefs (Jeffrey_Portfolio_Site_Plan.pdf and the resume PDF), built greenfield in d:\GitHub\Official Personal Website\, verified with Lighthouse, and pushed to GitHub. A prior draft (Personal Website First Draft\, plain HTML/CSS) was intentionally not reused.

Typography

  • Space Grotesk (variable) — display and body. Hero headline scales clamp(3rem, 10.5vw, 8.5rem), leading 0.95, tracking −0.03em. Statement/Contact headings clamp(2.5rem, 8vw, 6.5rem).
  • Space Mono — all labels, nav, dates, HUD, footer meta: 11px uppercase, wide tracking. This carries the reference site's technical character.

Signature elements (from the haoqi.design reference study)

  • HUD metadata — live ticking clock HH:MM:SS EST·NYC + Manhattan coordinates 40.7128°N × 74.0060°W in the hero; coordinates repeated in the footer copyright line. (Reference equivalent: "--:-- GMT+8 CN", "0001 X 0001 Y".)
  • Bracket controls — THEME[D]/THEME[L] toggle. (Reference: THEME[A], SOUND[|]; sound intentionally omitted per brief.)
  • Indexed project rows — mono 01/02 index, title up to 5xl, tag · year label, ↗ arrow, whole row is the link.
  • NYC stickers — four decorative SVGs (subway token, I♥NY-style mark, pigeon, water tower), slight rotation, reduced opacity, aria-hidden.
  • Background — static SVG film grain (feTurbulence data-URI) + two blurred radial-gradient blobs drifting on 40s/52s transform-only CSS keyframes. Zero per-frame JavaScript; disabled under reduced motion.
